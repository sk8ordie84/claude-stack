# Claude Quick Guide

## Main commands I actually use

### /video
Main video tool.
Use this when I upload an image and want the best video prompt fast.
It should choose between Runway and Kling automatically.

Example:
- Use the video skill on this image.
- Use the video skill and choose Runway.
- Use the video skill and choose Kling.

---

### /runway-45-video
Use this when I want a Runway Gen-4.5 prompt.
Best for:
- image-to-video
- elegant cinematic motion
- seamless transitions
- scale shift
- premium single-shot results

---

### /kling-30-video
Use this when I want a Kling 3.0 prompt.
Best for:
- 5-shot structure
- multishot progression
- stronger story flow
- fashion / product / narrative sequences

---

### /nano-banana-pro
Use this for image edit prompts.
Best for:
- outfit swaps
- identity lock
- object changes
- “change only X, keep everything else identical”

---

### /project-health-check
Use this when I want to check whether the setup is still healthy.
