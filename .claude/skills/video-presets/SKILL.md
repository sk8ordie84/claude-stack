# video-presets

Amaç: Runway Gen-4.5 ve Kling 3.0 için tekrar eden "show" presetlerini tek yerden yönetmek.

## Preset Listesi (kısa isim → içerik)

### runway:showtime
- Stil: modern, yüksek prodüksiyon, temiz geçişler
- Transition set: match cut, scale shift, seamless morph, dissolve, parallax, whip-pan (gerektiğinde)
- Kamera: static → slow push-in → micro-orbit (kontrollü)

### runway:scale-morph
- Scale shift + morph + seamless transition odaklı
- Negatif: flicker/warp/text kesin kapalı
- Kamera: sabit veya çok hafif push-in

### kling:guyritchie5
- 5-shot multishot (5×3s = 15s)
- Her shot: net aksiyon + sebep-sonuç
- Transition: hard cut / match cut, ritmik

### kling:fashionwalk
- Approach → Turn → Walk-away (loop hissi)
- Kumaş hareket kuralları, ayak yere basma gölgeleri
- Kamera: eye-level, stabil, editorial

### product:macro-hero-lifestyle
- 3-beat: Macro detail → Hero centered → Lifestyle usage
- Işık: soft studio / daylight, premium clean
