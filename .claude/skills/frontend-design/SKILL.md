Use when the user asks for web components/pages/apps with high design quality.

Guidelines:
- Understand the context (purpose, audience, constraints)
- Choose a bold aesthetic direction and stick to it
- Use distinctive typography and cohesive color themes (CSS variables)
- Add motion/micro-interactions deliberately (not everywhere)
- Create depth: gradient meshes, noise textures, patterns, overlap, shadows
- Avoid generic AI aesthetics (Inter/Roboto, purple gradients, generic layouts)

Implementation:
- Produce production-grade code (HTML/CSS/JS, React, etc.) with clean structure
- Match complexity to aesthetic (maximalism can require more code)
- Refine spacing, typography, and details
